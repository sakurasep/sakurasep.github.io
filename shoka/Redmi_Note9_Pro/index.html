# 文章简介

作者：上杉九月

排版与发布：上杉九月

# 前言

Note 系列一直是高质量千元机的典型代表，记得几年前 Note4X（625 版本），凭借着 14nm 和 8XA53 的神 U 骁龙 625，在日常使用和续航都有着很不错的表现，这也促使我再次回到米系阵营。
Note 系列的性能在千元机一直都有着不俗的表现，5 系的骁龙 636，7 系的 660 和 675，8 系的发哥力作 G90T。今年 Note9Pro 上搭载的骁龙 750G 同样是一款刀法精湛的 CPU。

这里说一下，因为各厂商发布新机的速度太快了，没精力每一部都云一云。本系列更新随缘，等到什么发布会再提起我的兴趣，我会考虑做一做。

# 正文

# CPU 骁龙 750G VS 骁龙 765G

骁龙 750G 采用 A77 x 2 + A55 x 6 的核心设计

参考 Geekbench 跑分 单核 645 多核 1963

骁龙 765G 采用 A76 x 2 + A55 x 6 的核心设计

参考 Geekbench（一加 Z）单核 612 多核 1955

765G 的两个大核心频率高低不同，应该归属于 1 + 1 + 6 架构设计。
750G 的 GPU 使用的是 Adreno 619，765G 的 GPU 用的是 Adreno 620。

765G 曼哈顿 3.0：54Fps，曼哈顿 3.1：37Fps
750G 曼哈顿 3.0：64Fps，曼哈顿 3.1：36Fps。

刀法精湛的膏通。。。可以说 765G 与 750G 有着几乎相同的性能表现，制程上还是 765G 的 7nm 较为稳定。

# 屏幕 120HZ LCD

中端机上这块 LCD120Hz 高刷屏幕是一个不错的卖点，居中打孔使得正面颜值极高看了手中的米 9，水滴的确没有居中打孔好看。目前还没使用过多档变速高刷，因为有了多档位高刷频率，那么不同场景的切换就是检验优化的良好场景。

# 拍照 大底高像素 1 亿

一亿像素终于优化到可以正常使用的地步了。我是很喜欢高像素，解析力和创作空间都是优点。发布会上展示的长卷裁切就是一个高像素裁切的很好地应用场景。双原生 ISO 对于拍照体验也是不小的提升，至于九合一就不好评价了。千元机同时有大底和高像素，理论上拍照体验是越级的。

# 细节

  • 双扬声器虽说不是对称双扬声器，但是千元机上有这个功能也会添色不少。

  • 线性马达刚需，一个好的马达 + 一个好的系统场景适配 = 优秀的日常使用体验。

  • 33W！！！中端机进一步下放充电技术

  • 生活防泼溅，抗摔抗砸，千元品质，做工很值得点赞。

  • NFC，红外，3.5mm 极其先进。

# 总结

CPU 性能足够强，日常使用没有问题，打打网游也不在话下；拍照越级体验，大底高像素以及调教的算法，发朋友圈效果很不错；续航久，充电快・・・・・・总的来说，依旧是一款风格独特的水桶机，是对去年 Note8Pro 的全面升级，1599 的起步价有点小贵，但是很多人购买中端机型预算很紧，可能加几百有更好的体验，受到预算的限制，加 100 都是很难的。Note9Pro 精准卡在了 1999 以下的一个空位档，在目前 K30U 很难买到的情况下，这个价位竞争力很大。
接下来，就是期待 2021 年全系 2k 的米 11 和 MIUI13 了。
